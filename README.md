## react-todo-app

### How to run

```bash
# Clone this repository
$ git clone https://github.com/korayguler/react-todo-app

# Go into the repository
$ cd react-todo-app


# Install dependencies
$ npm install

# Start the react
$ npm start

# start the json-server
$ npm run server

# Build
$ npm run build
```

### Tech Stack

- react
- json-server

### Prerequisite

- node
- npm
